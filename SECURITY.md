# Security Policy

Liquid Prnc Glass is a local WebGL/GLSL visual lab.

If you discover a security issue related to file handling, source loading, export, browser APIs, dependency vulnerabilities, or build tooling, please use GitHub private vulnerability reporting if it is enabled for the repository.

If private reporting is not enabled, open a minimal public issue that avoids exploit details and ask for a maintainer contact path.

## Scope

Relevant:

- unsafe file handling;
- export pipeline issues;
- dependency vulnerabilities;
- browser API misuse;
- build tooling problems.

Not relevant:

- visual artifacts;
- expected shader distortion;
- preset choices;
- performance tuning requests.

## Supported Versions

This project is currently in the `0.1.x` lab phase. Security fixes target the default branch until a release policy is formalized.
