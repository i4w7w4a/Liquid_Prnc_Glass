## Replacement Checklist

When regenerating assets:

1. Start this repository's dev server, not another project on the same port.
2. Confirm the page title is `Liquid Prnc Glass`.
3. Capture at least one clean-source frame.
4. Capture at least one shader-refraction frame.
5. Capture one field/region state.
6. Keep `hero.gif` small enough for README loading.
7. Do not commit large videos.
8. Use only rights-safe source media.

## Manual GitHub Step

Upload `social-preview.png` in repository settings as the social preview image.
