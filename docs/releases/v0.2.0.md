## What This Release Is

`v0.2.0` is the first broad laboratory release after the initial public shell. It turns the project from one edge-refraction preset into a fuller WebGL optics lab: selectable geometry, flow-field motion, source regions, color correction, source sizing, and export paths now share one preset contract.

## Highlights

- Shape Geometry Picker with regular and irregular SDF-backed shapes.
- Flow Field Controls that perturb the optical normal before signed IOR.
- Center-to-edge field, source regions, and smoother mask fading.
- Color controls for exposure, fine brightness, contrast, saturation, temperature, tint, and gamma.
- Corrected moving-media color decode with `sRGBTransferEOTF` and output through `linearToOutputTexel`.
- Source selection for image or motion sources, source sizing, PNG frame render, preview recording, and deterministic MP4 render.
- Generated integration brief with the full preset and readable grouped values.
- HTTPS public site link: https://liquid-prince.online/

## Preset Contract

The exported `Current preset` now includes every active setting:

- core optics;
- color correction;
- center field;
- shape geometry;
- optical flow;
- effect regions;
- render pixel ratio.

The brief generator includes the same JSON plus human-readable summaries for the major groups, so another engineer or agent can port the effect without guessing which sliders matter.

## Known Limits

- MP4 rendering still depends on browser support for WebCodecs/Mediabunny paths.
- Large source files and high render sizes can be slow on weaker GPUs.
- Flow is analytic and deterministic; fluid-simulation or ping-pong-buffer flow is still future work.
