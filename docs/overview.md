# Liquid Prnc Glass Overview

Liquid Prnc Glass is a standalone WebGL/GLSL optics lab for reusable liquid-glass refraction on video, image, and other visual sources.

Project links:

- Website: [http://liquid-prince.online/](http://liquid-prince.online/)
- Repository: [https://github.com/i4w7w4a/Liquid_Prnc_Glass](https://github.com/i4w7w4a/Liquid_Prnc_Glass)

## Core Statement

```text
Source texture in.
Optical field through GLSL.
Portable refraction out.
```

This project is not CSS glassmorphism, not a `backdrop-filter` trick, and not a landing page shell. The visible result is generated by a Three.js renderer and a GLSL fragment shader.

## What The Lab Does

- Loads a video or image source.
- Uploads the source as a GPU texture.
- Keeps a clean-source path for neutral settings.
- Computes a rounded-box SDF edge field.
- Can switch to a smooth center-to-edge optical field.
- Applies refraction, dispersion, edge darkening, and highlights.
- Lets the operator tune settings live.
- Serializes settings as JSON.
- Generates an integration brief for another project.
- Exports the rendered WebGL canvas through recording or deterministic MP4 rendering.

## What The Lab Refuses

- No CSS blur as a substitute for shader refraction.
- No decorative overlay pretending to be optics.
- No shader logic inside React UI.
- No website-clone scope creep.
- No new controls that do not map to a real renderer or shader behavior.

## Primary Audience

The project is useful for:

- creative coders;
- shader and WebGL engineers;
- frontend engineers moving visual effects into production UI;
- maintainers who need a controlled lab before porting the effect elsewhere.

## Success Criteria

A change belongs here when it makes the optical technique:

- more visually convincing;
- more controllable;
- more portable;
- easier to test;
- easier to explain.

If a change only makes the UI louder, it is probably noise.
