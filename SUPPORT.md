# Support

Use the right channel. It saves time and keeps the project readable.

## Reproducible Bugs

Use GitHub Issues:

```text
https://github.com/i4w7w4a/Liquid_Prnc_Glass/issues
```

Choose the closest template:

- Simple bug report;
- Bug report;
- Visual artifact;
- Export problem;
- Feature request.

For optical problems, include preset JSON, source type, browser, GPU, and a screenshot or clip.

## Questions And Tuning

Use GitHub Discussions:

```text
https://github.com/i4w7w4a/Liquid_Prnc_Glass/discussions
```

Use Discussions for:

- shader tuning;
- integration help;
- preset sharing;
- screenshots and clips;
- early ideas.

## Security

Read:

```text
SECURITY.md
```

Do not post exploit details in public issues.
